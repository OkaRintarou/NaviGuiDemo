# 不含导航的演示品
## 解释各文件夹用途
- .idea文件夹

  > 为idea自动生成的工程文件，没有用

- out文件夹

  > 编译和构建模块的输出产物

- src文件夹

  > 存放源码

- NaviGuiDemo.iml

  > idea工程

- README.md

  > 我写的说明书

## 大作业原理

- 寻路算法考虑使用迪杰斯特拉算法

- 将得到的路线分解为一帧一帧的指令用于动画

  > 所以一点都不难，就是加上一堆条件遍历

## 分工需求

- 画地图，用现成的也行，但是得打好网格，算好坐标，保证路线为直角转弯，不然寻路算法和动画都会非常麻烦
- 将地图转换为文件进行存储，也就是将每个单位的属性保存成文件
- 写文档
- 实现算法
